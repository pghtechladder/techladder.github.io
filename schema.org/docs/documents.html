Additional Resources

  • Schema.org blog

  • Schema.org Community Group

    Since April 2015, the W3C Schema.org Community Group is the main forum for schema collaboration, and provides the public-schemaorg@w3.org mailing list for discussions. Schema.org issues are tracked on GitHub and coordinated via the group.

    Earlier discussions can be found in W3C's public-vocabs group. There was also a google group used during 2011.

    Prior to 2014, the W3C Web Schemas group hosted a collection of proposals for additions and improvements to schema.org.
