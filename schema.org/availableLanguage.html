availableLanguage

A Schema.org Property
Thing > Property :: availableLanguage
  • Canonical URL: https://schema.org/availableLanguage
  • Check for open issues.
A language someone may use with or at the item, service or place. Please use one of the language codes from the IETF BCP 47 standard. See also inLanguage.

Values expected to be one of these types

  • Language
  • Text

Used on these types

  • ContactPoint
  • Course
  • LodgingBusiness
  • ServiceChannel
  • TouristAttraction
